# pwd
Print Working Directory
# ls
list files and directories in the current location 
# ls -l
Shows detailed information about files and directories, including permissions, owners, size and date (
# ls -a
Shows all files, including hidden files
# cd
Change the current directory
Ex: cd Documents
# mkdir 
Creates a new directory
# touch
Creates a new empty file
Ex: touch notes.txt
# mv 
Moves a file or directory. It can also be used to rename files
Ex: mv notes.txt documents/
    mv notes.txt linux-notex.txt
# .
Represents the current directory
# ..
Represents the parent directory (one level up)
Ex: cd .. ( moves to the directory above the current one)
# rm
Removes a file
Ex: rm notes.txt (Warning: deleted files cannot be easily recovered)
# rm -r
Removes a directory and all its contents recursively
Ex: rm -r old-project (Warning: This command deletes the directory and everything inside it)
